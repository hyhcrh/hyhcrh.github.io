Android Studio 添加 Vector Asset 不显示 / 只显示少量图标 怎么办 ?

  • 2025-02-09
  • 作者 hyhcrh
  • 440 字

本文最后编辑于 前，其中的内容可能需要更新。

删除 <SDK文件夹>/icons/material/icons_metadata.txt 就好了.

打开 SDK 文件夹 ( ~/Android/Sdk ), 进入 icons/material/, 删除 icons_metadata.txt 即可解决

附赠 Windows / macOS 默认 SDK 位置:

系统 默认 SDK 位置
Windows %USERPROFILE\Android\Sdk\
C:\Users\<用户名>\Android\Sdk\
macOS ~/Library/Android/sdk
/Users/<用户名>/Library/Android/sdk
